 Mlinzi Kids - GPS Tracking Button + Parent Dashboard

Track your child. Any phone. Anywhere.

Mlinzi Kids is a child safety platform built for Kenyan parents.
It combines a compact GPS tracking button with a web-based parent dashboard for real-time location monitoring, activity history, and danger alerts. No app download required.

[Mlinzi Kids](Mlinzi.png)

 The Problem
In Kenya, 7+ children are reported missing daily in crowded markets like Gikomba, Kongowea, and bus stations. 
Parents face 3 main issues:
1. Phones die, get stolen, or kids don't answer
2. GPS apps require data bundles + smartphones  
3. No way to see location history or set safe zones

The Solution
Mlinzi Button + Parent Portal
1. Button: Small GPS+GSM tracking chip kids carry in pocket, bag, or clip on shoe
2. SOS: Press 3 seconds → instant SMS alert to parents with location
3. Dashboard: Parents login on any browser to see live map, battery %, and activity history
4. No app needed: Works on Safaricom and Airtel. Even Nokia 3310 via SMS

 Features
- Real-Time GPS Tracking: Location updates every 10 seconds, accurate to 3-5m outdoors
- Parent Dashboard: Web portal accessible on phone or computer
- Danger Zone Alerts: Set safe zones around school/home. Get alert if child leaves
- Activity History: Track daily routes, patterns, and locations visited
- 7-Day Battery: Power-saving mode + low battery SMS alerts
- Waterproof IP67: Survives rain, swimming, daily use
- Multi-Parent Access: Mum, Dad, Guardian can all track one child
- SMS + App Mode: Works with or without smartphone
